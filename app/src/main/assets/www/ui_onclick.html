Introduction of OnClickListener

If you want to make a TextView, ImageView or any view clickable, You can declare the click event handler programmatically. To declare the event handler programmatically, create an View.OnClickListener object and assign it to the button by calling setOnClickListener (View.OnClickListener).


For example:

Button button = (Button) findViewById(
R.id.button_send);
button.setOnClickListener(
new View.OnClickListener() {
  public void onClick(View v) {
    // Do something
  }
});
