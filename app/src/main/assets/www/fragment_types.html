Types of Fragments

Basically fragments are divided as three stages as shown below.

• Define Fragments Statically − Defining Fragments Statically is using for hand hold devices like mobiles, here we can show only one fragment as a view. 
• Define Fragments Dynamically - Defining Fragments Dynamically is using with fragment transaction. We can move one fragment to another fragment. 
• List Fragments − List Fragments are fragments having special list view is called as list fragment.
