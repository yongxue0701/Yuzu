Example of Defining Fragment Statically

Adding two fragments in activity_main.xml.

activity_main.xml

<LinearLayout
  xmlns:android="http://schemas.
  android.com/apk/res/android"
  xmlns:tools="http://schemas.android.
  com/tools"
  android:id="@+id/LinearLayout1"
  android:layout_width="match_parent"
  android:layout_height="match_parent"
  android:orientation="horizontal" >

  <fragment
    android:id="@+id/fragment1" 
    android:name="YourFragmentName1" 
    android:layout_weight="1"
    android:layout_width="0dp" 
    android:layout_height=
    "match_parent"/>; 

  <fragment
    android:id="@+id/fragment2" 
    android:name="YourFragmentName2" 
    android:layout_weight="1"
    android:layout_width="0dp" 
    android:layout_height=
    "match_parent"/>; 
</LinearLayout>

Now, you can create Fragment1.java and Fragment2.java which have their own UI. Fragment will contribute its UI by using its own layout.

Fragment1.xml

<?xml version="1.0" encoding="utf-8"?>
<LinearLayout
  xmlns:android="http://schemas.
  android.com/apk/res/android"
  android:layout_width="match_parent"
  android:layout_height="match_parent"
  android:orientation="vertical" >

  <TextView
    android:id="@+id/textView1"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:text="I m fragment ONE"
    android:gravity="center"
    android:background="#5eff6a"
    android:textAppearance="
    ?android:attr/textAppearanceLarge" />
</LinearLayout>

After you have defined it this way, donâ€™t try to replace, delete, etc. with a FragmentTransaction! You obtain a reference to the fragment in Java using the FragmentManager.

Fragment1.java

package com.example.staticfragments;

import android.annotation.SuppressLint;
import android.app.Fragment;
import android.os.Bundle;
import android.view.LayoutInflater;
import android.view.View;
import android.view.ViewGroup;

public class Fragment1 extends Fragment {
  public View onCreateView(
  LayoutInflater inflater, ViewGroup vg,
  Bundle savedInstanceState) {
    return inflater.inflate
    (R.layout.fragment1, vg,false);
  }
}
