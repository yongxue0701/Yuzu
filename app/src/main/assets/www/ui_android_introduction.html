Introduction of Android

What is a Mobile App?
Applications can reside in Server or in Client.

Resident in Server:
Web applications optimized for mobile devices (we don’t call these mobile apps).
• Method 1: Detect browser capabilities
  - WURFL (framework used by Facebook)
  for mobile device detection.
  - ASP.NET can be configured to use WURFL
  - can group devices based on capabilities
  and serve accordingly
• Method 2: Use Responsive Web Design

Resident in Client:
Native Mobile Apps: Swift in iOS, Java in Android
Hybrid Mobile Apps: HTML5 + JavaScript + CSS3

Note: In this course, we will pay more attention on development of Native Mobile Applications.
